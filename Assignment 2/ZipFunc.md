```python
# iterable 1 
list1 = [11, 21, 31]

# iterable 2 
list2 = ['ab', 'bc', 'ca']

# iterable 3 
list3 = ['xy', 'yz', 'za']

# zipping the three iterables with zip() function 
zipped = zip(list1, list2, list3)

# The list() function is creating a list of zip which is ordered 
result = list(zipped)

# printing the output
print(result)

```
